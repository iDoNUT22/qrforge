# 🧬 QR Forge Pro

**QR Forge Pro** is a powerful and modern QR code generator with built-in logo integration, customizable styling, and instant download support. Designed for clean design workflows and reliable QR output with high scan accuracy.

---

👉 **Live Demo:**  
https://your-live-link-here.com

---

## ✨ Features

- 🔗 Generate QR codes from URLs or text instantly  
- 🖼️ Embed center logos inside QR codes  
- 🎨 Fully customizable colors (foreground & background)  
- 📏 Adjustable logo size with real-time preview  
- 📱 Drag & drop logo upload support  
- 🛡️ High error correction level (H-level) for scan reliability  
- ⬇️ One-click PNG download  
- ⚡ Live QR regeneration on input changes  
- 💻 Clean, responsive, modern UI  

---

## 🧠 How It Works

1. Enter a URL or text  
2. Upload a logo (optional)  
3. Customize colors and logo size  
4. Generate your QR code instantly  
5. Download and use anywhere  

---

## ⚠️ Notes

- Very large logos may reduce scan readability  
- Uses high error correction to support logo embedding  
- Best results with centered, simple logos  

---

## 🚀 Tech Stack

- HTML5 Canvas  
- Vanilla JavaScript  
- QRCode.js Library  
- Modern CSS (Flexbox + Gradients)
