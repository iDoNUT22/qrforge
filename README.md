# 🧬 QR Forge Pro — Advanced QR Code Generator

A modern, feature-rich QR code generator built for **fast creation, logo embedding, and high scan reliability**.

Designed with a clean UI and real-time customization for developers, designers, and everyday users.

## ✨ Features

### 🔗 QR Code Generation

* Generate QR codes instantly from URLs or text
* Real-time QR updates as you type
* High-quality output for reliable scanning

### 🖼️ Logo Integration

* Embed center logos inside QR codes
* Drag & drop logo upload support
* Adjustable logo sizing with live preview

### 🎨 Customization

* Foreground and background color control
* Instant visual feedback while editing
* Clean, modern UI for fast design workflow

### ⚡ Export & Performance

* One-click PNG download
* High error correction (H-level) for scan stability
* Fast rendering and smooth updates

---

## 🧠 How It Works

1. Enter a URL or text
2. Upload a logo (optional)
3. Customize colors and logo size
4. QR code updates instantly
5. Download and use anywhere

---

## ⚠️ Notes

* Large logos may reduce scan readability
* High error correction improves logo support
* Best results: simple, centered logos
